
골이 들어갈 때마다 들어가는 시간과 팀을 적는다.

농구 경기는 정확히 48분 동안 진행됨.

각 팀이 몇 분동안 이기고 있었는지 출력.

N 입력

둘째 줄부터 N개의 줄에 득점 정보가 주어짐...
- 득점한 팀 번호 + " " + 득점한 시간
- 팀 번호는 1 또는 2
- 득점한 시간 MM:SS(분:초)
- 분과 초가 한 자리일 경우 첫째자리가 0이다
- 분은 0보다 크거나 같고, 47보다 작거나 같으며, 초는 0보다 크거나 같고, 59보다 작거나 같다
(0 <= min <= 47), (0 <= second <= 59)
